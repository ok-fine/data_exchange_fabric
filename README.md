nodejs：用于系统搭建

# 说明
## 目录
├── back_end
│   ├── application
│   ├── chaincode
│   ├── config
│   ├── data-network
├── front_end

front_end: 前端代码
back_end: 应用代码
* application: 系统实现（nodejs）
* chaincode: 链码设计
* config: 配置文件夹
* data-network: fabric区块链网络搭建
